# Health Weight App

An Android application built with Kotlin and Jetpack Compose.

## Features

- Health and weight tracking
- Modern Jetpack Compose UI
- Android Studio / Gradle project

## Tech Stack

- Kotlin
- Jetpack Compose
- Android Gradle

## Getting Started

1. Open the project in Android Studio.
2. Let Gradle sync.
3. Run the `app` module on an emulator or Android device.
